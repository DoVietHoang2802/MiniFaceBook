# Frontend Master (React/Next.js)

## 🎯 Mục tiêu
Xây dựng giao diện người dùng mượt mà, chuẩn UI/UX và Type-safe tuyệt đối.

## 🛠 Quy tắc vàng
- **State Management:** Ưu tiên **TanStack Query** cho Server State. Hạn chế lạm dụng `useEffect`.
- **Styling:** Sử dụng Tailwind CSS + shadcn/ui.
- **Validation:** Sử dụng **Zod** để đồng bộ kiểu dữ liệu với Backend.
- **Performance:** Tối ưu hóa render và sử dụng Lazy loading cho các component nặng.

## 🔄 Mandatory Post-Task Workflow (BẮT BUỘC)
Sau mỗi Task, phải tự động:
1. Cập nhật **PROGRESS.md** và **ROADMAP.md**.
2. **Cập nhật Internal Skills** nếu có kiến thức mới.
3. Kiểm tra tính đồng bộ của toàn bộ file tài liệu (.md).
4. Báo cáo các file đã cập nhật.

## 💡 Tư duy Senior
- Giao diện đẹp là chưa đủ, phải nhanh và dễ tiếp cận (Accessibility).
- Luôn xử lý các trạng thái Loading và Error một cách tinh tế.
